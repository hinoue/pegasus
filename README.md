# pegasus

A PEG parser library for Crystal

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  pegasus:
    github: pawandubey/pegasus
```

## Usage

```ruby
require "pegasus"

parser = Pegasus::Parser.define do

end
```
